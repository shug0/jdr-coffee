# Historical Taxonomy Reference

## Periods

| Code | Label | Range | Description |
|------|-------|-------|-------------|
| `prehistory` | Préhistoire | -3M to -3300 | Before writing |
| `ancient` | Antiquité | -3300 to 476 | First civilizations → Fall of Rome |
| `medieval` | Moyen Âge | 476 to 1492 | Fall of Rome → Discovery of Americas |
| `renaissance` | Renaissance | 1300 to 1600 | European cultural renewal |
| `early-modern` | Époque moderne | 1492 to 1789 | Great discoveries → French Revolution |
| `modern` | Époque contemporaine | 1789 to 1945 | French Revolution → End WW2 |
| `contemporary` | Histoire récente | 1945 to 2025 | Since end WW2 |

## Usage Notes

- **Period**: Choose primary period, note if spans multiple
- **Regions**: Can be multiple (e.g., trade routes)
- **Themes**: Multiple themes common (e.g., economics + warfare for weapon prices)
- **Validation**: All codes must exist in index.json taxonomy before use
