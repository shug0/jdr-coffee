# Historical Research Sources - Quick Reference

## Source Reliability Tiers

### ⭐⭐⭐⭐⭐ High Reliability
- **JSTOR** (jstor.org) - Peer-reviewed academic articles
- **HAL** (hal.science) - French open academic archive
- **DOAJ** (doaj.org) - Directory of Open Access Journals
- **Gallica** (gallica.bnf.fr) - BnF digital library, primary sources
- **Internet Archive** (archive.org) - Historical documents, books

### ⭐⭐⭐⭐☆ Medium Reliability
- **Google Scholar** - Academic search, verify sources
- **Wikipedia** - Use for context + extract academic references
- **Isidore** (isidore.science) - French SHS meta-search
- **BASE** (base-search.net) - Multi-disciplinary academic search

### ⭐⭐⭐☆☆ Use With Caution
- Specialized historical sites (medievalcoinage.com, etc.) - IF well-sourced
- University course materials - IF by credible professors
- Museum websites - IF primary sources cited

## Search Strategy

1. **Start specific**: "medieval sword prices England 1300-1400"
2. **Use French for European topics**: "prix épées médiévales"
3. **Extract Wikipedia references**: Mine academic citations
4. **Cross-validate**: Minimum 2 independent sources

## Reliability Checklist

✅ Author with credentials (PhD, professor, museum curator)
✅ Published by academic institution or peer-reviewed journal
✅ Primary sources cited
✅ Recent publication (post-2000 for syntheses) OR primary source
✅ Multiple concordant sources

❌ Anonymous author
❌ No sources cited
❌ Commercial site
❌ Blog without academic backing
